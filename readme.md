
##  FreeWebpilotInGPT35

FreeWebpilotInGPT35 要解决的问题是ChatGPT Web版非付费用户也可以使用访问外部数据导入到ChatGPT上下文处理,这个功能是在ChatGPT 4.0 + Plugin(Webpilot)版本才可以使用
